# Dynamic Web-Based Quiz & Examination Platform

A responsive, single-page online quiz application featuring user registration, persistent login sessions, and an asynchronous per-question countdown timer.

## Features
* **User Authentication:** Complete Sign In and Registration system that persists data locally.
* **Dynamic Quiz Flow:** Questions load seamlessly one by one without page reloads.
* **Timed Challenges:** A strict 10-second countdown timer per question that automatically handles timeouts.
* **Instant Results:** Immediate score calculation and summary screen upon completion.

## Tech Stack
* **Frontend:** HTML5, CSS3, JavaScript (ES6)
* **Storage Layer:** Web Storage API (`localStorage`)
